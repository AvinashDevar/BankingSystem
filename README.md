# BankingSystem
A simple Banking Management System developed to handle basic banking operations like account creation, deposit, withdrawal, and balance inquiry. This project demonstrates core concepts of programming, database management, and user interaction.


---
## 🚀 Key Features

- Create new bank account
- Deposit money
- Withdraw money
- Transfer money
- Check account balance
- View account details
- Delete account
- Secure login system 

---

## 🛠️ Technologies Used
| Technology      | Purpose                                    |
| --------------- | ------------------------------------------ |
| Core Java       | Business logic and application development |
| Hibernate (JPA) | ORM framework for database interaction     |
| PostgreSQL      | Relational database                        |
| Maven           | Dependency management and build tool       |

---

## ⚙️ How to Run the Project

#### 1.Clone the repository
- git clone https://github.com/AvinashDevar/BankingSystem.git
#### 2.Open the project
- Import into Eclipse or IntelliJ IDEA
#### 3.Set up the database
- Install PostgreSQL
- Create a database:  CREATE DATABASE banking;
#### 4.Configure Hibernate
- Open hibernate.cfg.xml
- Update database credentials
#### 5.Build the project
#### 6.Run the application
- Run the main class (Controller) from your IDE
